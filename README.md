# Green Light Group — Landing Page

React + TypeScript public marketing website for **Green Light Group**.

## Requirements

- Node.js 20+
- Backend API running (optional for dynamic content)

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Environment

```env
VITE_API_URL=http://127.0.0.1:8765/api
```

## Pages

Home, About, Services, Industries, Logistics, Banking Consulting, Certificates, News, Contact, Request Quote, Privacy, Terms.

## Build

```bash
npm run build
npm run preview
```
